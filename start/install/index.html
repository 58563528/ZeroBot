    快速安装 #

    安装Go #

    首先需要安装Go(支持版本1.15+), 你可以在 Go官网 或者 Go中国镜像站 (国内用户推荐) 找到对应的安装包

    下载和导入项目 #

    本项目使用了GO111MOUDULE, 你需要使用Go模块集成管理(如果不会就用Goland帮你解决吧)

    使用下面指令下载源码

     go get github.com/wdvxdr1123/ZeroBot

    然后你就可以在项目中导入ZeroBot了!

    import "github.com/wdvxdr1123/ZeroBot"
